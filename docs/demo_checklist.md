# 演示与调试清单

由于当前无法立刻插入硬件，建议正式录制前按下面顺序排查。

## UART 必做演示

1. 只给主机上电，用串口工具确认每 `50ms` 有一帧数据输出。
2. 慢慢旋转电位器，观察帧中角度从 `0°` 到 `180°` 连续变化。
3. 给从机上电，确认 `PA6` PWM 周期为 `20ms`，脉宽在 `0.5ms..2.5ms`。
4. 连接主机 `PA9` 到从机 `PA10`，两板共地。
5. 旋转电位器，录制舵机同步转动，重点展示不卡顿、无明显抖动、断开通信后回到 90°。

## CAN 加分演示

1. 两块 TJA1050 模块分别接主从 STM32。
2. 确认 CANH/CANL 未接反，两板和模块全部共地。
3. 主机发送标准帧 `0x321`，DLC 为 `4`。
4. 从机只接收 `0x321`，解析角度并控制舵机。
5. 录制从 UART 线切换到 CAN 线后的同步效果。

## 常见问题

| 现象 | 优先检查 |
| --- | --- |
| 舵机乱抖 | 舵机是否独立 5V 供电、是否与 STM32 共地、PWM 周期是否为 20ms |
| 角度方向反了 | 交换电位器 3.3V/GND 两端，或在代码中用 `1800 - angle` |
| UART 无反应 | 主机 `PA9` 是否接从机 `PA10`，波特率是否都是 9600 |
| CAN 无反应 | CANH/CANL、TJA1050 供电、波特率、滤波 ID `0x321` |
