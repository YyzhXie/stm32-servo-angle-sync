# 接线说明

完整硬件连接图见 [硬件连接图](hardware_connection.md)。

## UART 必做版

| 信号 | 主机 STM32F103C8T6 | 从机 STM32F103C8T6 | 说明 |
| --- | --- | --- | --- |
| UART TX/RX | `PA9 TX` | `PA10 RX` | 主机发送角度到从机 |
| GND | `GND` | `GND` | 两块板必须共地 |
| 电位器中间脚 | `PA0/ADC1_IN0` | - | 两侧脚接 `3.3V` 和 `GND` |
| 舵机信号 | - | `PA6/TIM3_CH1` | 输出 50Hz PWM |
| 舵机供电 | - | 外部 `5V` | 舵机电源建议独立供电并与 STM32 共地 |

如果需要双向串口调试，可以额外连接主机 `PA10` 到从机 `PA9`，但本题必做同步只需要主机
TX 到从机 RX。

## CAN 进阶版

| 信号 | 主机 | 从机 | 说明 |
| --- | --- | --- | --- |
| CAN_TX | `PA12` | `PA12` | 分别接本板 TJA1050 TXD |
| CAN_RX | `PA11` | `PA11` | 分别接本板 TJA1050 RXD |
| CANH | TJA1050 CANH | TJA1050 CANH | 总线高 |
| CANL | TJA1050 CANL | TJA1050 CANL | 总线低 |
| GND | 主机 GND | 从机 GND | 两板与两个 CAN 模块必须共地 |

短线桌面测试时通常可以直接连 CANH/CANL；若通信不稳定，在总线两端加 120 欧终端电阻。

## SG90 舵机脉宽

| 角度 | 高电平脉宽 | 占空比 |
| --- | --- | --- |
| `0°` | `0.5ms` | `2.5%` |
| `90°` | `1.5ms` | `7.5%` |
| `180°` | `2.5ms` | `12.5%` |
